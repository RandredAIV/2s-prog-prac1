# RU

# Порядок сборки проекта на языке C++

## Описание

Данная практическая работа посвящена изучению порядка сборки проекта на языке C++ 
с использованием командной строки.

В рамках работы рассмотрены этапы сборки программы: препроцессинг, компиляция, 
ассемблирование и линковка. Также изучен процесс компиляции и запуска программ 
на языке Java в операционных системах Windows и Linux.

Дополнительно продемонстрировано создание bat-файла для автоматизации сборки.

---

## Цели работы

- Изучение этапов сборки проекта на языке C++
- Освоение работы в командной строке Windows и Linux
- Изучение процесса компиляции и запуска программ на языке Java
- Ознакомление с основными командами операционных систем
- Автоматизация сборки проекта с помощью bat-файла

---

## Задание 1 — Сборка проекта на C++ (Windows)

- Написание программы на языке C++
- Демонстрация этапов сборки:
  - Препроцессинг
  - Компиляция
  - Ассемблирование
  - Линковка
  - Запуск исполняемого файла
- Анализ промежуточных файлов (*.ii, *.s, *.o)

---

## Задание 2 — Второй язык программирования (Windows)

- Выбор языка Java
- Компиляция исходного файла
- Получение байт-кода (*.class)
- Запуск программы через JVM
- Создание jar-архива

---

## Задание 3 — Сборка проекта на C++ (Linux)

- Работа в терминале Linux
- Выполнение всех этапов сборки
- Демонстрация владения командами Linux

---

## Задание 4 — Второй язык программирования (Linux)

- Компиляция и запуск Java-программы
- Работа в среде Linux

---

## Задание 5 — Создание bat-файла

- Создание bat-файла
- Автоматизация процесса сборки
- Добавление комментариев к этапам сборки

---

## Используемые языки программирования

- C++
- Java

## Вывод

В результате работы был изучен полный цикл сборки программ

# EN

# C++ Project Build Process

## Description

This practical work is devoted to studying the build process of a C++ project 
using the command line.

The work covers all stages of program building: preprocessing, compilation, 
assembling and linking. The compilation and execution process of Java programs 
in Windows and Linux operating systems is also demonstrated.

Additionally, the creation of a batch file for build automation is presented.

---

## Objectives

- Study the stages of building a C++ project
- Practice working in Windows and Linux command line
- Learn how to compile and run Java programs
- Become familiar with basic OS commands
- Automate the build process using a batch file

---

## Task 1 — C++ Build Process (Windows)

- Write a C++ program
- Demonstrate the build stages:
  - Preprocessing
  - Compilation
  - Assembling
  - Linking
  - Running the executable file
- Analyze intermediate files (*.ii, *.s, *.o)

---

## Task 2 — Second Programming Language (Windows)

- Choose Java as a second language
- Compile the source file
- Generate bytecode (*.class)
- Run the program using JVM
- Create a jar archive

---

## Task 3 — C++ Build Process (Linux)

- Work in Linux terminal
- Perform all build stages
- Demonstrate knowledge of Linux commands

---

## Task 4 — Second Programming Language (Linux)

- Compile and run a Java program
- Work in Linux environment

---

## Task 5 — Creating a Batch File

- Create a .bat file
- Automate the build process
- Add comments for each build stage

---

## Programming Languages

- C++
- Java

---

## Conclusion

As a result of this practical work, the full build cycle of C++ and Java programs 
in Windows and Linux operating systems was studied and demonstrated.
